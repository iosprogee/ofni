#### [vi]()

| keys   | action |
| :---   | :---   |
|<kbd>ctrl</kbd> + <kbd>v</kbd> | column mode  |
|<kbd>d</kbd>                   | delete       |
|<kbd>i</kbd>                   | edit mode    |
|<kbd>esc</kbd>                 | command mode |
|<kbd>/</kbd><stringtosearch> <kbd>enter</kbd>| search       |
|<kbd>:</kbd> <kbd>w</kbd> <kbd>q</kbd> | save & exit  |
