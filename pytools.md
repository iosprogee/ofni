#### CMSeeK (py3)
```
  **install**  
	git clone https://github.com/Tuhinshubhra/CMSeeK
	cd CMSeeK
	pip3 install -r requirements.tx
  **use**  
  python cmseek.py -u <url> --user-agent Mozilla 5.0
  cat cat Result/<url>/cms.json
```
